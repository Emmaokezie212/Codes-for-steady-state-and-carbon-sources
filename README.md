This project contains the data generated for the manuscript entitled: The syntrophic synergy between Clostridium carboxidivorans and Clostridium beijerinckii during butanol production 

All simulations were performed using COBRApy (https://opencobra.github.io/cobrapy/) in Python 3.10, with the Gurobi (https://www.gurobi.com/) linear programming solver. 

For more information about the models read the “Models information.docx”
